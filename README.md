# 🏋️‍♂️ GymNerds - Gestão de Treinos e PRs

Este é um aplicativo de gestão de treinos desenvolvido com **React Native** e **Expo**. O foco do app é permitir que atletas e entusiastas de musculação organizem suas fichas de treino, selecionem exercícios e acompanhem seus recordes pessoais (PRs).

## 🚀 Tecnologias Utilizadas

- **Framework:** [Expo (SDK 51)](https://expo.dev/)
- **Linguagem:** TypeScript
- **Roteamento:** Expo Router (File-based routing)
- **Estilização:** Styled Components / React Native Styles
- **Gerenciamento de Estado:** Zustand (Stores para Fichas e PRs)
- **Ícones:** Lucide React Native / Expo Icons

## 🛠️ Requisitos de Ambiente

Devido às atualizações recentes do ecossistema Node.js, este projeto requer:
- **Node.js:** v20.x ou superior (Testado na v24)
- **Gerenciador de pacotes:** npm ou yarn
